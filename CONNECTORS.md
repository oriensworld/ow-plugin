# Connectors

## Local MCP Server

This plugin uses a **local MCP server** instead of a remote connector.
The server runs on your machine via `npx`.

| Category | Server | How it runs |
|----------|--------|-------------|
| System info & echo | `@oriensworld/ow-server` | Local stdio via `npx` (auto-installed) |

### Requirements
- Node.js >= 18
- No API keys or authentication needed
